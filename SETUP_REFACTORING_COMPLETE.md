## 🚀 Usage Guide

### **Running the Setup Wizard:**

```bash
# Windows
python setup.py

# Linux/Mac
python3 setup.py
```

### **Step-by-Step Process:**

**Step 1: Initialize Project**
- Creates all required directories
- Creates `.gitkeep` files
- Sets up folder structure

**Step 2: Install Dependencies**
- Reads `requirements.txt`
- Installs all Python packages
- Shows progress indicator
- Handles errors gracefully

**Step 3: Server Configuration**
- Username (default: admin)
- Password (min 6 characters)
- Server URL (default: http://localhost:5000)

**Step 4: MT5 Configuration**
- Browse for `terminal64.exe`
- Validates file exists
- Auto-sets instances directory

**Step 5: Email Configuration (Optional)**
- Toggle email notifications
- Sender email & app password
- Recipient email(s)
- All fields optional

**Step 6: Generate Configuration**
- Validates all inputs
- Backs up existing .env (if any)
- Generates secure keys (SECRET_KEY, WEBHOOK_TOKEN)
- Creates comprehensive .env file
- Offers to start server

## 🔄 Migration Notes

### **For Existing Users:**

**If you have an existing `.env` file:**
1. The wizard will detect it
2. Offers to backup before overwriting
3. Creates timestamped backup (`.env.backup.YYYYMMDD_HHMMSS`)
4. All old values preserved in backup

**Manual Migration:**
```bash
# Backup your current .env
cp .env .env.backup

# Run setup wizard
python setup.py

# Copy any custom values from backup if needed
```
